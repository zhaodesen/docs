    # 浏览器样式

    浏览器样式（更恰当的名称是用户代理样式）是元素尚未设置样式时浏览器应用在它身上的默认样式

    # 用户样式

    大多数浏览器都允许用户定义自己的样式表，这个功能用的人不多

    # 样式层叠

    浏览器显示元素时求索 CSS 属性值的次序： 元素内嵌样式 > 文档内嵌样式 > 外部样式 > 用户样式 > 浏览器样式

    元素内嵌样式 , 文档内嵌样式 , 外部样式 合称 作者样式

    定义在用户样式表中的样式称为用户样式

    有浏览器定义的样式称为浏览器样式

    将样式属性值标记为重要，浏览器会优先考虑

    a {
    color: red !important;
}
